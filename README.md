# Front-Saas-Zappé
# 🚀 Zappé — Landing Page para Plataforma de Pedidos Online

Uma landing page moderna e responsiva desenvolvida em HTML e CSS para apresentação da plataforma **Zappé**, uma solução inteligente para gerenciamento de pedidos online, cardápios digitais e automação de atendimento para restaurantes e estabelecimentos de alimentação.

---

## 📋 Sobre o Projeto

O Zappé é uma plataforma voltada para digitalização de restaurantes, lanchonetes, pizzarias, deliverys e outros negócios alimentícios.

Esta landing page foi criada para apresentar os principais benefícios da plataforma, incluindo:

- Cardápio digital
- Gestão de pedidos
- Atendimento automatizado por IA
- Dashboard de vendas
- Gestão de entregas
- Pagamentos integrados
- Planos comerciais

## 🚀 Como Executar

### Opção 1 — Abrir Diretamente

Basta abrir o arquivo:

```bash
index.html
```

em qualquer navegador moderno.

---

### Opção 2 — VS Code + Live Server

1. Instale a extensão Live Server.
2. Abra o projeto no VS Code.
3. Clique com o botão direito no arquivo:

```text
index.html
```

4. Selecione:

```text
Open with Live Server
```

O navegador será aberto automaticamente.

## 🔒 Segurança Recomendada

Ao transformar a landing page em uma aplicação real:

- HTTPS obrigatório
- WAF (Cloudflare ou AWS WAF)
- Autenticação JWT
- Criptografia de dados
- Rate Limiting
- Monitoramento de logs
- Backup automático
- Proteção contra XSS
- Proteção contra SQL Injection
- Proteção CSRF
